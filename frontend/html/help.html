Frequently Asked Questions

What is the difference between Search Mode and Create Mode?

Search Mode helps you find existing questions and answers in our knowledge base using AI-powered vector search. Create Mode allows you to submit new questions that will be added to our database and can be answered by administrators or the community.

How long does it take for my question to be answered?

Questions submitted through Create Mode are reviewed by our team. Response times vary depending on the complexity of the question and current queue volume. Most questions receive answers within 24-48 hours during business days.

Why doesn't my search return any results?

This could happen if your question is very unique or if you're using very specific terminology. Try rephrasing your question, using more general terms, or breaking down complex questions into simpler parts. If you still don't find answers, switch to Create Mode to submit your question.

What is the Admin panel and who can access it?

The Admin panel is a management interface for reviewing and answering submitted questions, managing the knowledge base, and monitoring system activity. Access is restricted to authorized administrators who have been provided with valid API keys.

How does the AI search work?

Askly uses advanced vector search technology powered by AI embeddings. When you ask a question, it's converted into a mathematical representation (vector) that captures its semantic meaning. This allows us to find similar questions even if they use different words but have the same intent.

How do I enable dark mode?

Click the moon/sun icon in the navigation bar on any page to toggle between light and dark themes. Your preference will be saved automatically and applied across all pages. Askly also respects your system's dark mode preference by default.
